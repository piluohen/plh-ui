# Plh-ui

## 使用

```bash

## 安装 npm 包

npm i -S plh-ui

## 项目引入css

import "plh-ui/lib/index/style.css";

## 项目main.js引入组件, 全局引入

import PlhUi from 'plh-ui'

Vue.use(PlhUi)
```

## 构建命令

```bash
# 安装依赖
npm install

# 启动本地服务（默认test环境接口） localhost:8080
npm run dev

# 打包docs
npm run build

# 打包组件库
npm run lib
```
